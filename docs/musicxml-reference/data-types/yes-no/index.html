MusicXML 4.0 > MusicXML reference > Data types > yes-no

yes-no data type

The yes-no type is used for boolean-like attributes. We cannot use W3C XML Schema booleans due to their restrictions on expression of boolean values.

Base type: token

Allowed values

yes True.
no False.
