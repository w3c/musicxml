The <creator> element

Parent element: <identification>

The <creator> element describes the creators of the score. This is similar to the Dublin Core creator element. The MusicXML format does not use the creator / contributor distinction from Dublin Core.

Content

string

Attributes

Name Type Required? Description
type token No Distinguishes different creative contributions. Thus there can be multiple <creator> elements within an <identification> element. Standard values are composer, lyricist, and arranger. Other values may be used for different types of creative roles. This attribute should usually be used even if there is just a single <creator> element.
