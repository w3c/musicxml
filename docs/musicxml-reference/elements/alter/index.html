The <alter> element

Parent element: <pitch>

The <alter> element represents chromatic alteration in number of semitones (e.g., -1 for flat, 1 for sharp). Decimal values like 0.5 (quarter tone sharp) are used for microtones.

Content

semitones

Attributes

None.

Examples

This element is used in the following examples:

<accidental>, <alter> (Microtones), <alter> (Semitones), <backup>, <chord> (Multiple Stop), <forward>, <grouping>, <ipa>, Tutorial: Après un rêve, Tutorial: Chopin Prelude, Tutorial: Tablature
