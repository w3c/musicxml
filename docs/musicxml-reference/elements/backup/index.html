The <backup> element

Parent elements: <measure> (partwise), <part> (timewise)


The <backup> and <forward> elements are required to coordinate multiple voices in one part, including music on multiple staves. The <backup> element is generally used to move between voices and staves. Thus it does not include <voice> or <staff> elements. Duration values should always be positive, and should not cross measure boundaries or mid-measure changes in the <divisions> value.

Content

In this order

  • <duration> (Required)
  • In this order
    • <footnote> (Optional)
    • <level> (Optional)

Attributes

None.

Examples

This element is used in the following examples:

<backup>, <except-voice>, <forward>, <glissando> (Multiple), <pedal> (Lines), <pedal> (Symbols), <voice>, Tutorial: Après un rêve, Tutorial: Chopin Prelude, Tutorial: Percussion
